# Bonds-Personal-Crm
Bonds is a personal CRM I built to manage relationships the way professionals manage clients — tracking contacts, meetings, birthdays, and follow-ups in one clean interface. Built entirely with vanilla HTML, CSS, and JavaScript, no frameworks or backend required. Data persists locally in the browser, so it works completely offline. The app features a relationship warmth scoring system that surfaces drifting connections and suggests who to reach out to next, alongside networking insights that give a bird's-eye view of your social health.
